<h3>VGA Sync Generator for Arduino</h3>
<p>Tested on the Sparkfun Pro Micro 16MHz 5V version but should work with any Leonardo based board.
<p>Hardware connections:
<br>Pin 9 outputs H sync
<br>Pin 5 outputs V sync</p></br></p>
<p>For instructions to upload to your SparkFun Pro Micro 5V 16MHz board:
<br>https://support.arduino.cc/hc/en-us/articles/4733418441116-Upload-a-sketch-in-Arduino-IDE</p>
<p>Instructions to add board definitions for Sparkfun Pro Micro 5V board:
<br>In the Arduino IDE go to preferences.
<br>In the "Additional Boards Manager URLs" text entry box paste: 
<br>https://raw.githubusercontent.com/sparkfun/Arduino_Boards/main/IDE_Board_Manager/package_sparkfun_index.json
<br>Click "OK"
<br>In the main Arduino IDE window select the second tab from the top - the boards manager icon.
<br>Select "Sparkfun AVR Boards" and click "Install"</p>
